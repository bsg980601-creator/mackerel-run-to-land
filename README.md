# Mackerel: Run to Land (고등어 육지 대탈출)

공연 〈박첨지 고민상담소〉 EP2용 원버튼 아케이드 게임. Made by **A moment lab**.

## ▶ 바로 플레이
**https://bsg980601-creator.github.io/mackerel-run-to-land/**

## 조작
| 키 | 동작 |
|---|---|
| **SPACE / ENTER** | 탭 = 위로 헤엄 · 0.6초 홀드 = 파워대시(그물 파괴) |
| 그물 트랩에 걸리면 | **스페이스 연타**로 빨간 게이지를 0까지 깎아 탈출 |

- 하트 3개 = 도전 횟수. 다 잃으면 GAME OVER → 스페이스로 재도전
- 심해 → 중층 → 수면 → 갯벌 4개 존, 마지막엔 육지 도착
- 점수·최고기록(브라우저 저장) — 팀원끼리 경쟁 가능

## 오퍼레이터 키 (공연용)
`S` 시작 · `R` 리셋 · `T` 튜토리얼 스킵 · `F` 비상 피날레 · `1~4` 존 점프

## 로컬 실행
저장소를 받아 `index.html`을 브라우저로 열거나, `python3 -m http.server` 후 접속.
